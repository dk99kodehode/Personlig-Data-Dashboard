⚙️ Functions

- Legg til exercises med timer / minutter
- Har en progress bar som virker
- Lagrer Exercise,Duration, Favorites og timestamps i Local data Storage

Hvordan du skal bruke den

1. Trykk på top feltet for å velge exercise som var gjort
2. Tast inn minutten
3. Rediger / delete som du vill ut i fra behov
4. Sjå progress baren fylle opp 😉
